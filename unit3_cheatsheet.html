Looping in Ruby and Javascript

Ruby

To loop through some code a certain number of times

3.times do
  puts "Aloha!"
end
# => Prints "Aloha" 3 times

To loop through and array and execute some code on the values

numbers = [0, 1, 2, 3, 4]
numbers.each do |x|
  p x * 2
end
# => Prints 0, 2, 4, 6, 8 

numbers.each { |x| p x }
# => Prints 0, 1, 2, 3, 4  

# Another way to loop through an array
numbers = [0, 1, 2, 3, 4]
numbers.length.times do |i|
  puts numbers[i]
end
# => Prints 0, 1, 2, 3, 4

To loop from one number up to another number. Can also use "downto".

1.upto(5) do |i|
  puts i
end
# => Prints 1, 2, 3, 4, 5

Using "while" and "until" to loop through an array

numbers = [0, 1, 2, 3, 4]
i = 0
while i < numbers.length
  puts numbers[i]
  i += 1 
end

numbers = [0, 1, 2, 3, 4]
i = 0
until i >= numbers.length
  puts numbers[i]
  i += 1 
end
# => Both print 0, 1, 2, 3, 4

JavaScript

To loop through and array and execute some code on the values

numbers = [0, 1, 2, 3, 4]
for (var i = 0; i <= numbers.length; i++ ) { 
  console.log(numbers[i])
} 
// => Prints 0, 1, 2, 3, 4

Using "while" to loop

var i=0;
while (i<=10) {
  console.log(i)
  i++;
}
// Prints the values from 0 to 10

To loop through properties of an object using "for...in"

var letters = {a: "A", b: "B", c: "C"};
var letter = '';
for (letter in letters) {
	console.log(letter);
}
// Prints ['a','b','c'];
